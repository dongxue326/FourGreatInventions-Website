# 重命名GitHub仓库指南

## 📝 步骤一：在GitHub上重命名仓库

1. **访问你的仓库**：https://github.com/dongxue326/FourGreatInventions-Website
2. **点击 "Settings"**（设置）
3. **滚动到页面最底部**，找到 **"Repository name"**（仓库名称）部分
4. **输入新名称**：`multiculturefestive-China-2025-barbieri`
5. **点击 "Rename"**（重命名）按钮
6. GitHub会显示确认信息，点击确认

**注意**：重命名后，旧的URL会自动重定向到新的URL，但建议更新所有链接。

---

## 🔄 步骤二：更新本地远程仓库地址

在GitHub上重命名完成后，运行以下命令更新本地配置：

```bash
git remote set-url origin https://github.com/dongxue326/multiculturefestive-China-2025-barbieri.git
```
